# 🎲 Simple Dice Game

## 🔍 About
This is a simple command-line dice rolling game built using Python. When you run the game, it simulates rolling a 6-sided dice and prints the result.

## 🛠️ Technologies Used
- Python 3.x
- Built-in `random` module
